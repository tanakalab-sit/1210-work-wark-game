# ミニゲーム集

## 概要

このプロジェクトは、単一のHTMLファイルで動作する、2種類のミニゲームを収録したブラウザゲームです。

1.  **風船クリッカーゲーム:** サポートキャラクターを選んで目標スコアを目指すクリッカーゲームです。
2.  **数字選びゲーム:** コンピューターと数字の読み合いで対戦する、シンプルな運試しのゲームです。

## 遊び方

1.  `index.html` ファイルをウェブブラウザで開きます。
2.  最初に表示される画面で、プレイしたいゲームモードを選択してください。

### ① 風船クリッカーゲーム

- **準備:**
    1.  目標スコアを入力します。
    2.  10種類の個性的な動物キャラクターから、一緒に戦うサポートキャラを1体選びます。
    3.  「ゲーム開始！」ボタンを押します。

- **操作方法:**
    - **マウスクリック:** 画面中央の風船をクリックするとスコアが増えます。
    - **キーボード入力:** キーボードで好きな文字を入力し、`Enter`キーを押すと、入力した文字数に応じたボーナススコアが加算されます。

### ② 数字選びゲーム

- **ルール:**
    - 3つのライフを持ってゲームが始まります。
    - 1から4までの数字の中から、コンピューターが選び**そうにない**数字を1つ選択します。
    - コンピューターと同じ数字を選んでしまうとライフが1つ減ります。
    - ライフが0になる前に、できるだけ多くのスコアを獲得することを目指します。

### 隠しコマンド (共通)

- `qqq`: 強制的にエンドロールを開始します。
- `qq`: （クリッカーゲーム中のみ）チートモードのパネルを開閉します。

## 工夫した点

- **単一ファイル構成:**
  HTML、CSS、JavaScriptのすべてを1つの`index.html`ファイルにまとめることで、取り扱いや配布を容易にしています。

- **多彩なゲームプレイ:**
  単純なクリッカーゲームに加え、思考タイプの異なる「数字選びゲーム」を実装し、飽きさせない工夫を凝らしました。

- **リプレイ性:**
  クリッカーゲームでは、それぞれ異なる効果を持つ10種類のサポートキャラクターから1体を選ぶことができ、毎回違った戦略で楽しめます。

- **豊かな演出:**
  - CSSアニメーションによる動的な背景や、クリック時のパーティクルエフェクト。
  - 目標達成後に表示される運勢占い機能。
  - ゲームの締めくくりとして流れる、遊び心のあるエンドロール。

- **隠し要素:**
  通常のプレイに加えて、`qq`や`qqq`といった隠しコマンドを用意し、発見する楽しみを加えました。
